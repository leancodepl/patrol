## Tips and tricks

### Running a test many times

Sometimes, you might want to run a test many times. In such case, we've got
these one-liners for you:

**Bash**

```bash
for i in {1..10}; do maestro drive --target integration_test/my_test.dart; done
```

**PowerShell**

```powershell
1..10 | ForEach-Object { maestro drive --target integration_test/my_test.dart }
```
