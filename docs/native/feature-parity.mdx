## Feature parity

We strive for high feature parity across iOS and Android, but unfortunately it's
impossible to reach 100%.

| **Feature**               | **Android** | **iOS** |
| ------------------------- | ----------- | ------- |
| Press home                | ✅          | ✅      |
| Press back                | ✅          | ✅      |
| Open any app              | ✅          | ✅      |
| Open notifications        | ✅          | ❌      |
| Tap on notification       | ✅          | ❌      |
| Open quick settings       | ✅          | ❌      |
| Toggle dark mode          | ✅          | ❌      |
| Toggle Wi-Fi              | ✅          | ❌      |
| Toggle Celluar            | ✅          | ❌      |
| Toggle Location           | ✅          | ❌      |
| Toggle Bluetooth          | ❌          | ❌      |
| Tap                       | ✅          | ✅      |
| Double tap                | ✅          | ✅      |
| Enter text                | ✅          | ✅      |
| Handle permission dialogs | ✅          | ✅      |
